UPLOAD TO GITHUB:
1. Open your redesign repository.
2. Choose Add file > Upload files.
3. Upload index.html, styles.css, script.js, README.txt, and the assets folder.
4. Replace the existing files when prompted.
5. Commit the changes.

The files must be in the repository root, not inside an extra folder.
