# Lucas Leitenberger Website

## Uploading to GitHub Pages

1. Extract this ZIP file.
2. Upload every file and folder inside it to the root of your GitHub repository.
3. In `index.html`, replace every instance of `YOUR_EMAIL_HERE` with your real email address.
4. Commit the changes.
5. In GitHub, open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select your main branch and the root folder, then save.

The site includes:

- Vy Truong headshot

- Pennsylvania and Ohio market sections
