# Lucas Leitenberger Recruiting Website

A responsive, single-page recruiting website built with plain HTML, CSS, and JavaScript. No build step is required.

## Before publishing

1. Open `index.html`.
2. Search for `lucasleitenberger@gmail.com`.
3. Replace it with your preferred business email.
4. Review the opportunity lists and update them as needed.

## Publish with GitHub + Netlify

1. Create a new GitHub repository.
2. Upload all files from this folder, including the `assets` folder.
3. In Netlify, choose **Add new site → Import an existing project**.
4. Connect GitHub and select the repository.
5. Leave the build command blank.
6. Set the publish directory to `.` if Netlify asks.
7. Click **Deploy site**.

Because this is a static website, Netlify should publish it immediately.

## Local preview

Double-click `index.html`, or open the folder in VS Code and use the Live Server extension.

## Files

- `index.html` — page content
- `styles.css` — design and responsive layout
- `script.js` — mobile navigation and scroll animations
- `assets/lucas-leitenberger.jpg` — profile image
- `netlify.toml` — Netlify configuration
